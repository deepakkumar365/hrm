## 🎭 Visual Effects

### **Box Shadows**
```css
/* Container Shadow */
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12);

/* Subtle depth, professional appearance */
```

### **Hover Effects**
```css
/* Table Row Hover */
.pay-table tbody tr:hover {
    background: #F9F9F9;
    transition: background 0.2s ease;
}

/* Smooth transition on hover */
```

### **SVG Wave Pattern**
```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1200 15">
  <path fill="#C7E3E6" d="M0,10 Q300,0 600,10 T1200,10 L1200,15 L0,15 Z"/>
</svg>

/* Creates smooth wave at top of footer */
```
