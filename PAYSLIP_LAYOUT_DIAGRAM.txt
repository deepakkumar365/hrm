┌─────────────────────────────────────────────────────────────────────────────────────┐
│ ALLOWANCES BREAKDOWN (from employee.allowances)                                     │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  Total Allowances = employee.allowances                                             │
│                                                                                     │
│  HRA              = Total Allowances × 0.40  (40%)                                  │
│  DA               = Total Allowances × 0.30  (30%)                                  │
│  Travel Allowance = Total Allowances × 0.20  (20%)                                  │
│  Special Allowance= Total Allowances × 0.10  (10%)                                  │
│                                                                                     │
│  Example: If employee.allowances = S$ 2,000.00                                      │
│    HRA              = S$ 2,000.00 × 0.40 = S$ 800.00                                │
│    DA               = S$ 2,000.00 × 0.30 = S$ 600.00                                │
│    Travel Allowance = S$ 2,000.00 × 0.20 = S$ 400.00                                │
│    Special Allowance= S$ 2,000.00 × 0.10 = S$ 200.00                                │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────┐
│ DEDUCTIONS BREAKDOWN (from payroll.other_deductions)                                │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  Total Other Deductions = payroll.other_deductions                                  │
│                                                                                     │
│  Professional Tax = Total Other Deductions × 0.40  (40%)                            │
│  ESI Contribution = Total Other Deductions × 0.30  (30%)                            │
│  Insurance Premium= Total Other Deductions × 0.20  (20%)                            │
│  Other Deductions = Total Other Deductions × 0.10  (10%)                            │
│                                                                                     │
│  Example: If payroll.other_deductions = S$ 300.00                                   │
│    Professional Tax = S$ 300.00 × 0.40 = S$ 120.00                                  │
│    ESI Contribution = S$ 300.00 × 0.30 = S$  90.00                                  │
│    Insurance Premium= S$ 300.00 × 0.20 = S$  60.00                                  │
│    Other Deductions = S$ 300.00 × 0.10 = S$  30.00                                  │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────┐
│ GROSS SALARY CALCULATION                                                            │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  Gross Salary = Basic Pay + Overtime Pay + Allowances + Bonuses                    │
│                                                                                     │
│  Example:                                                                           │
│    Basic Pay      = S$ 5,000.00                                                     │
│    Overtime Pay   = S$   500.00                                                     │
│    Allowances     = S$ 2,000.00                                                     │
│    Bonuses        = S$ 1,000.00                                                     │
│    ─────────────────────────────                                                    │
│    Gross Salary   = S$ 8,500.00                                                     │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────┐
│ TOTAL DEDUCTIONS CALCULATION                                                        │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  Total Deductions = Employee CPF + Income Tax + Other Deductions                    │
│                                                                                     │
│  Example:                                                                           │
│    Employee CPF      = S$ 1,000.00                                                  │
│    Income Tax        = S$   500.00                                                  │
│    Other Deductions  = S$   300.00                                                  │
│    ──────────────────────────────                                                   │
│    Total Deductions  = S$ 1,800.00                                                  │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────┐
│ NET SALARY CALCULATION                                                              │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│  Net Salary = Gross Salary - Total Deductions                                       │
│                                                                                     │
│  Example:                                                                           │
│    Gross Salary      = S$ 8,500.00                                                  │
│    Total Deductions  = S$ 1,800.00                                                  │
│    ──────────────────────────────                                                   │
│    Net Salary        = S$ 6,700.00                                                  │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
