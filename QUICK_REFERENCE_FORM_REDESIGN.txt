CSS CLASSES REFERENCE:
================================================================================

.form-grid
   • Grid container for form fields
   • Handles responsive column distribution
   • 16px gap between items

.form-grid-item
   • Individual form field wrapper
   • Wraps label, input, and help text
   • Flex column for proper stacking

.form-grid-item.full-width
   • Spans entire width (all columns)
   • Used for: Address textarea, Profile Image upload
   • grid-column: 1 / -1

.form-grid-item.half-width (optional)
   • Spans 2 columns on 4-column layout (≥1200px)
   • For future use if needed

COMMON CUSTOMIZATIONS:
================================================================================

Change gap between fields:
   .form-grid { gap: 12px; }  /* Default: 16px */

Change focus border color:
   .form-grid-item .form-control:focus {
       border-color: #FF6B6B;  /* Change from teal to red */
   }

Change columns for specific breakpoint:
   @media (min-width: 1400px) {
       .form-grid { grid-template-columns: repeat(5, 1fr); }
   }

Change input padding:
   .form-grid-item .form-control {
       padding: 10px 12px;  /* Default: 12px 12px */
   }

================================================================================
TROUBLESHOOTING:
================================================================================

Q: Fields showing in 2 columns instead of 4 on desktop?
A: Hard refresh (Ctrl+Shift+R), clear browser cache

Q: Focus border not showing teal color?
A: Check browser developer tools, verify CSS loaded
   Might be conflicting styles - check CSS precedence

Q: Mobile showing 2 columns instead of 1?
A: Check viewport meta tag, disable browser zoom, test actual device

Q: Full-width fields not spanning entire width?
A: Verify class="form-grid-item full-width" is present in HTML
   Check CSS rule: .form-grid-item.full-width { grid-column: 1 / -1; }

Q: Generate button not aligned with Employee ID?
A: Verify .input-group uses flexbox layout
   Check CSS: .form-grid-item .input-group { display: flex; }
