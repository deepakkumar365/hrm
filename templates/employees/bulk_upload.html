How to Use Bulk Upload
  1. Download Template: Click the "Download Template" button below to get the Excel template with all required fields.
  2. Fill Employee Data: Open the template and fill in employee information starting from row 3.
    • Fields marked with * are mandatory
    • Mandatory fields are highlighted in YELLOW
    • Follow the format specified in row 2 (descriptions)
  3. Save the File: Save your Excel file after filling in the data.
  4. Upload: Click "Upload Excel" button and select your filled template.
  5. Review Results: The system will process the file and show you the results.

Important Notes
  • Unique Fields: Employee ID, Email, and NRIC must be unique. Duplicate values will be rejected.
  • Date Format: All dates must be in YYYY-MM-DD format (e.g., 2024-01-15).
  • Company Code: Must match an existing company code in the system.
  • User Role: Must be one of: Super Admin, Admin, HR Manager, Manager, or User.
  • Default Password: All new employees will be assigned the default password and must reset it on first login.
  • Profile Images: Profile images cannot be uploaded via bulk upload. They must be added individually after employee creation.
  • Validation: The system will validate all data before creating employees. Invalid rows will be skipped with error messages.
